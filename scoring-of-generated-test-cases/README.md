# Scoring 
This folder contains the scoring of all test cases generated by ChatGPT-5.2, Gemini 3.1 Pro, and Claude Sonnet 4.6. The scoring was done by one reviewer, and followed metrics derived from established test case quality attributes, OWASP Top 10 best practices, and externally defined acceptance criteria. 

# Transparency
To promote transparency, consistency and validity in test case quality scoring, the test cases have been mapped to each other across models if applicable.
