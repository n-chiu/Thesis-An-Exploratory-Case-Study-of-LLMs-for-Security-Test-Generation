# (22) Broken Access Control + Injection

## Title: Untrusted data handling

## ID: 22

## User Story:
As a Software company Customer, I want the application to handle untrusted data securely.

## Acceptance Criteria:

Untrusted file data should be handled in a secure manner, with data from untrusted sources stored outside the web root and with limited permissions.

Verify that files obtained from untrusted sources are validated to be of expected type.

Untrusted file data submitted to the application is not used directly with file I/O commands, particularly to protect against path traversal, local file include, file mime type, and OS command injection vulnerabilities.

Untrusted data is not used within inclusion, class loader, or reflection capabilities to prevent remote/local file inclusion vulnerabilities.

Untrusted data is not used within cross- domain resource sharing (CORS) to protect against arbitrary remote content.

The application code does not execute uploaded data obtained from untrusted sources

## Edge Cases:

File uploads with disguised extensions containing scripts.

Path traversal attempts.

Large file uploads causing denial of service.

Archive files (ZIP, TAR) containing malicious nested paths (“zip slip”).

Files with embedded malware or scripts.

Use of uploaded files in downstream processes (e.g. image processing libraries with vulnerabilities).

Unicode or encoded filenames bypassing validation.

CORS misconfiguration allowing malicious origins.

## Constraints:

Strict allowlist for file types, extensions, etc.

Store untrusted files outside web root with minimal permissions.

Rename uploaded files (no user-controlled file names).

Enforce file size limits and scan for malware.

Never execute or interpret uploaded files.

Validate and sanitize all file paths.

Disable dynamic inclusion (no runtime loading from user input).

Strict CORS policy (no wildcard for sensitive endpoints).

Use sandboxing for file processing if needed.

## Vulnerability Category:

Broken Access Control + Injection
