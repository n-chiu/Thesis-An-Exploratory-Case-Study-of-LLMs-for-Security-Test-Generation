# Generated Test Cases
This folder contains all generated test cases from ChatGPT-5.2, Gemini 3.1 Pro, and Claude Sonnet 4.6. 

The generated test cases are separated by user story ID and categorized into its corresponding model.

The test cases were generated using the following prompt.

```text
Generate test cases for the following user story. Present the output as a table using the standardized test case template 
with the following fields: 
Test Case
Title/Scenario
Preconditions
Test Steps
Expected Results
Environment Specification
  
As a Software company Customer, I want the mobile client to observe the principle of "least privilege”
```
> Note: The user story above is part of the prompt and is provided as an example. Replace it with the target user story being tested when running the prompt.

# Standardized Test Case Template
All generated test cases conform to the standardized test case template. However, it is worth mentioning that test cases generated by Claude Sonnet 4.6 show slight misalignment with the template by introducing additional columns.
